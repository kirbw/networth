# Annual Finance Tracker

A lightweight web app for tracking yearly:

- Total annual income
- Charitable giving
- Net worth

## Features

- Save one record per tax year
- Update an existing year by re-submitting the form with the same year
- Visual trend graph using Chart.js
- Table view for all saved data
- Delete individual years or clear all records

## Run locally

Because this is a static web app, you can open `index.html` directly in your browser.

Or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
